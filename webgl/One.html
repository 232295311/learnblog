# One

# 2020.3.5

这是我的第一篇博客，来记录点啥好呢？ 说起来部署好这个可真是废了我好大一份心思呢，我用了两天的时间才一步步搞定这个东西，期间遇见了好多Bug，现在来回忆一下吧。

# 起步

前两天突然看到别人也有了自己的博客，哎呀作为一个大三的程序员了怎么可以连自己的博客都没有呢，说起来真是让人笑话呢，所以就去查了资料，最终决定用vuepress来做，参考的就是右边→参考的网站。 总而言之就是配置好环境，然后跟着官方文档弄就行了。

# 经历

刚起步的我真是啥也不懂呀，这也是我第一次接触到markdown格式，所以我完全不懂为什么他们的README要那样写啊，明明就没有html的内容为什么还能显示出那么好看的界面，于是就复制黏贴复制黏贴哈哈。 OK，那么写完配置和.md文件后，通过'npm run dev'，在本地也能够成功预览了，那就该部署了！！然而这也是我第一次用git上传文件夹呀哈哈哈o(////▽////)q，所以我都不懂，就去查查查，跟着教程一步步输命令。 OK，那我也上传好了，欣喜地打开网址，啊怎么显示不完全？啊于是就开始了咱的debug之旅。

# DeBug

说是debug其实只是重新做了一遍而已。注意config.js里面base需要写上自己仓库的名字就行了。但是明明我上一个项目里有写的呀，可还是渲染不成功，鬼知道为什么╮(╯▽╰)╭。

# 心路感想

终于来到了我喜欢的心路感想时间，其实我也只想写这时间，因为上面根本没啥好记录的呀，摔。 这是第一次用Vuepress，真的很庆幸做了这个决定。让我了解到了markdown是如此新奇的玩意儿，让我第一次亲手使用了git的一系列操作。在这之前我是对这两个方面完全一窍不通的，特别是git，到大三的人了每次都是叫别人帮我传，这也是终于我自己传了一次。也遇见了特别多的bug，昨晚搞到一点多，今早重新仔仔细细地做了一遍就好了，我也是不懂为什么。 总之通过这两天对这两个方面都有很多了解了。接下来要做的就是把用vue把首页弄得好看些，就像之前在编网站那样~加油吧
